Abstract of  KR20190084850 (A)


Tooltip
 

According to one aspect of the present invention, provided is an apparatus which includes: a memory circuit die configured to store a look-up table converting first data into second data; a logic circuit die including combination logic circuits configured to receive the second data; and an optical via configured to be connected between the memory circuit die and the logic circuit die and to transmit the second data from between the memory circuit die and the logic circuit die.
